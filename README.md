# Codveda Data Analytics Internship Projects

This repository contains the projects completed during my Data Analytics Internship at Codveda Technology.

The objective of these projects is to practice real-world data analysis tasks such as data cleaning, exploratory data analysis, machine learning, and data visualization.

## Projects Included

1. Iris Dataset Analysis
   - Exploratory Data Analysis
   - Data Visualization
   - Correlation Analysis

2. House Price Prediction
   - Data Cleaning
   - Feature Analysis
   - Linear Regression Model

3. Customer Churn Prediction
   - Classification Model
   - Random Forest Algorithm
   - Model Evaluation

4. Sentiment Analysis
   - Text Processing
   - Sentiment Classification

5. Stock Price Analysis
   - Time Series Visualization
   - Trend Analysis
